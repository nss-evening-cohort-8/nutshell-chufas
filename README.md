# Nutshell - Chufas

### Description
Nutshell is a dashboard for people to organize their daily weather, events, articles, and chat messages.

![homepage](./screenshots/nutshell.png)

### Technologies Used
* HTML
* JavaScript
* Bootstrap
* jQuery
* Axios
* Firebase
* CRUD
* Webpack
